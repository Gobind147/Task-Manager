## Setup 
1) create MongoDB Atlas account database url 
2) npm install
3) npm setup
4) npm start

## Features :
1) The backend is built with node.js , express.js Framework ,and MongoDb Database
2) Generic Crud Api (Create / Read / Update / Delete)


## Dependencies used
1. dotenv: 8.6.0
2. express: 4.17.1
3. mongoose: 5.13.22
4. nodemon: 2.0.7
